## Hardware
- cpu 13900F
- gpu : rx570
- memory : crucial ddr5-5200 64gb(32gb*2)(overclock 5600)
- cpu cooler :  valkyrie c280w
- SSD : WD black sn850 500g*2
- motherboard : asus strix b760-i
- case : JONSBO VR3
- power supply : Corsair SF600 80PLUS Gold
